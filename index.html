Sobre o Lucas

Uma pessoa que tem a motivação e a felicidade de trazer a solução de necessidades por meio do desenvolvimento de sistemas. C#

C# é uma linguagem de programação, desenvolvida pela Microsoft como parte da plataforma .NET, para mais informações clique aqui

A línguagem que estudo desde o começo na faculdade e a que atuo no mercado atualmente, me sinto confortável em dizer que é a com mais experiência e expertise atuando com projetos desktop a web.

.NET Core

.NET Core é um framework livre e de código aberto para os sistemas operacionais Windows, Linux e macOS. É um sucessor de código aberto do .NET Framework. Para mais informações clique aqui.

Framework ao qual posso dizer junto do C# que tenho mais expertise, apesar de estudar a mais tempo o .NET "tradicional", o .NET Core é o framework ao qual apliquei todo meu conhecimento de maneira profissional trazendo resoluções mais complexas.

.NET Framework

O .NET Framework é uma iniciativa da empresa Microsoft, que visa uma plataforma única para desenvolvimento e execução de sistemas e aplicações. Todo e qualquer código gerado para .NET pode ser executado em qualquer dispositivo que possua um framework de tal plataforma. Para mais informações clique aqui.

Framework que tenho um longo estudo seja a nível acadêmico e também com estudo autodidata, desenvolvendo simples projetos em WinForms, e outros projetos pessoais somente para estudos.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. Para mais informações clique aqui.

Linguagem que tive a introdução na faculdade a 2 anos atrás, apesar de um conhecimento básico me sinto confortável em utiliza-lá já que a utilizo no trabalho atual além de ter alguns cursos onlines.

JQuery

jQuery é uma biblioteca de funções JavaScript que interage com o HTML, desenvolvida para simplificar os scripts interpretados no navegador do cliente. Para mais informações clique aqui.

A ferramenta que mais utilizo junto do JavaScript para poder criar paginas Web ou WebAPIs.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript. Para mais informações clique aqui.

Tenho conhecimento básico porém é uma ferramenta que utilizo bastante, pois quando necessito criar uma página do "zero" para acelerar o processo de desenvolvimento normalmente recorro a ela, então me sinto confortável para trabalhar com a mesma.

Azure

O Microsoft Azure é uma plataforma destinada à execução de aplicativos e serviços, baseada nos conceitos da computação em nuvem. Para mais informações clique aqui.

Uma ferramenta que comecei a conhecer no estágio, porém continuo os estudos por fora desde então, já que utilizo muito atualmente, pois os projetos Web estão armazenados na App Services, além de utilizarem o SQL no Azure, Blobs para armazenamento, e CDN para a página estática. Atualmente além de utilizar o portal Azure utilizo as ferramentas Azure Storage Explorer e Azure Storage Emulator.

SQL Server

O Microsoft SQL Server é um sistema gerenciador de Banco de dados relacional desenvolvido pela Sybase em parceria com a Microsoft. Esta parceria durou até 1994, com o lançamento da versão para Windows NT e desde então a Microsoft mantém a manutenção do produto. Para mais informações clique aqui.

Banco de dados ao qual tenho mais conhecimento, estudo e utilizo o mesmo desce o início da minha jornada, ao qual atualmente utilizo o SQL Server Management Studio para gerenciamento dos DBs.

Git

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.. Para mais informações clique aqui.

O sistema de versionamento ao qual estou habituado a trabalhar, sempre utilizei o Git, e fiz o versionamento ou pelo GitHub ou GitLab.
